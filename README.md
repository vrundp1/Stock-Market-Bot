# Stock Market Bot

This is a stock market bot created in Python. It utilizes the Telegram and Yahoo Finance API to provide information on the stock companies you inquire about.

# Requirements
1. Download the .py file
2. Save it in a folder on Desktop (ex. Stock Market bot)
3. Make sure you have VS Code and Python
4. Install the packages on VS code - type in the terminal (python -m pip install python-telegram-bot pandas yfinance)

## Steps to Run
1. Create a bot with BotFather on Telegram.
2. Copy the Token (Note: DO NOT SHARE YOUR TOKEN WITH ANYONE ELSE)
3. On Powershell type - cd "C:\Users\YOUR_NAME\Desktop\Stock market bot"
4. Then type - $env:TELEGRAM_BOT_TOKEN="PASTE_YOUR_TOKEN_HERE"
5. Then run - python stock_bot.py
6. You should see - Bot running... Ctrl+C to stop.

## Chat
Open your Bot chat and send /start
Then fetch any stock data by running /stock Company (ex. /stock netflix)
